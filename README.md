# Wiring

To connect a NodeMCU 1.0 to the RFID-RC522:

| ESP8266-12E | NodeMCU | RFID-RC522 |
|:-----------:|:-------:|:----------:|
| GPIO12      | D6      | MISO       |
| GPIO13      | D7      | MOSI       |
| GPIO14      | D5      | SCK        |
| GPIO04      | D2      | SDA        |
| GPIO05      | D1      | RST        |
